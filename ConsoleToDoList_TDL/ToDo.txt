Задача: Разработать консольное приложение на C# для управления списком задач (To-Do List).

КОДОВОЕ НАЗВАНИЕ ПРОЕКТА TDL, в консоли будет вызываться командой "tdl"

Условия:

1. Приложение должно позволять пользователю:
    +++DONE добавлять новые задачи, 
    +++DONE отмечать задачи как выполненные, 
    +++DONE удалять задачи,
    +++DONE просматривать список всех задач.
2. Каждая задача должна иметь:
    +++DONE название (текст)
    +++DONE статус (выполнено/не выполнено).
3. Используйте класс для представления задачи (например, Task), который будет содержать 
    +++DONE свойства для имени задачи,
    +++DONE статуса задачи
4. Для хранения списка задач используйте массив или список (на ваше усмотрение).
5. Реализуйте методы:
    +++DONE добавления, 
    +++DONE удаления, 
    +++DONE изменения статуса задачи,
    +++DONE вывода списка задач на экран.
6. Взаимодействие с пользователем должно осуществляться через консоль (ввод/вывод).
7. Убедитесь, что программа корректно обрабатывает ввод пользователя 
    (например, при удалении задачи, которая не существует или ввода некорректных значений в переменные).
8. Данные должны сохраняться, если приложение будет перезапущенно
